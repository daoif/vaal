## 硬约束（Hard Constraints）

必须满足，违反则任务失败。

### 代码规范
- [ ] 所有函数必须有 JSDoc 注释

- [ ] 禁止使用 `var`，使用 `const` 或 `let`

### 类型安全
- [ ] 禁止使用 `any` 类型（如使用 TypeScript）
- [ ] 所有公共 API 必须有类型定义

### 测试要求
- [ ] 单元测试覆盖率 ≥ 80%
- [ ] 所有边界条件必须测试

### 安全要求
- [ ] 敏感数据必须加密
